# Locale & Language Checker

Reports current locale settings, available locales, encoding configuration, and language environment issues.

Part of the [Dargslan Linux Toolkit](https://dargslan.com) - Professional Linux & DevOps tools.

## Installation

```bash
pip install dargslan-locale-check
```

Or install the complete toolkit:

```bash
pip install dargslan-toolkit
```

## Usage

```bash
dargslan-locale report
dargslan-locale list
dargslan-locale encoding
```
