# dargslan-modprobe-check

Checks loaded kernel modules, blacklists, and modprobe configuration

## Install

```bash
pip install dargslan-modprobe-check
```

## Usage

```bash
modprobe-check
modprobe-check --help
modprobe-check --json
```
