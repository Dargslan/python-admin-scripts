# NTP Time Sync Checker

Verifies NTP daemon status, clock offset measurement, stratum level, and time synchronization configuration.

Part of the [Dargslan Linux Toolkit](https://dargslan.com) - Professional Linux & DevOps tools.

## Installation

```bash
pip install dargslan-ntp-check
```

Or install the complete toolkit:

```bash
pip install dargslan-toolkit
```

## Usage

```bash
dargslan-ntpchk report
dargslan-ntpchk offset
dargslan-ntpchk status
```
