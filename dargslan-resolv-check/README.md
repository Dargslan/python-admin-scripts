# DNS Resolver Checker

Validates resolv.conf configuration, nameserver reachability, search domain settings, and DNS resolution performance.

Part of the [Dargslan Linux Toolkit](https://dargslan.com) - Professional Linux & DevOps tools.

## Installation

```bash
pip install dargslan-resolv-check
```

Or install the complete toolkit:

```bash
pip install dargslan-toolkit
```

## Usage

```bash
dargslan-resolv report
dargslan-resolv test
dargslan-resolv servers
```
