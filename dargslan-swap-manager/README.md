# Swap Space Analyzer

Reports swap usage, swappiness settings, swap file vs partition details, and memory pressure indicators.

Part of the [Dargslan Linux Toolkit](https://dargslan.com) - Professional Linux & DevOps tools.

## Installation

```bash
pip install dargslan-swap-manager
```

Or install the complete toolkit:

```bash
pip install dargslan-toolkit
```

## Usage

```bash
dargslan-swap report
dargslan-swap usage
dargslan-swap swappiness
```
