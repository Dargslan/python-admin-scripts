# Environment Variable Auditor

Scans environment variables for sensitive data exposure, PATH issues, duplicate entries, and shell profile problems.

Part of the [Dargslan Linux Toolkit](https://dargslan.com) - Professional Linux & DevOps tools.

## Installation

```bash
pip install dargslan-env-audit
```

Or install the complete toolkit:

```bash
pip install dargslan-toolkit
```

## Usage

```bash
dargslan-envaudit report
dargslan-envaudit secrets
dargslan-envaudit path
```
