# SSH Hardening Checker

Audits sshd_config for security best practices including root login, password auth, key-based auth, port settings, and protocol versions.

Part of the [Dargslan Linux Toolkit](https://dargslan.com) - Professional Linux & DevOps tools.

## Installation

```bash
pip install dargslan-ssh-hardening
```

Or install the complete toolkit:

```bash
pip install dargslan-toolkit
```

## Usage

```bash
dargslan-sshd report
dargslan-sshd keys
dargslan-sshd config
```
