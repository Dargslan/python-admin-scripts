# Sysctl Parameter Auditor

Audits kernel parameters for network tuning, memory management, and security hardening against best practices.

Part of the [Dargslan Linux Toolkit](https://dargslan.com) - Professional Linux & DevOps tools.

## Installation

```bash
pip install dargslan-sysctl-audit
```

Or install the complete toolkit:

```bash
pip install dargslan-toolkit
```

## Usage

```bash
dargslan-sysctl report
dargslan-sysctl network
dargslan-sysctl security
```
