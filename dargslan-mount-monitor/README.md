# Mount Point Monitor

Tracks mounted filesystems, mount options analysis, read-only detection, stale NFS mount checking, and tmpfs usage.

Part of the [Dargslan Linux Toolkit](https://dargslan.com) - Professional Linux & DevOps tools.

## Installation

```bash
pip install dargslan-mount-monitor
```

Or install the complete toolkit:

```bash
pip install dargslan-toolkit
```

## Usage

```bash
dargslan-mounts report
dargslan-mounts options
dargslan-mounts stale
```
