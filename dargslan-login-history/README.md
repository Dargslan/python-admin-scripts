# Login History Analyzer

Parses wtmp/btmp for login history, failed login attempts, session duration tracking, and suspicious activity detection.

Part of the [Dargslan Linux Toolkit](https://dargslan.com) - Professional Linux & DevOps tools.

## Installation

```bash
pip install dargslan-login-history
```

Or install the complete toolkit:

```bash
pip install dargslan-toolkit
```

## Usage

```bash
dargslan-logins report
dargslan-logins failed
dargslan-logins sessions
```
