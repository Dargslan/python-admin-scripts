# Audit Log Analyzer

Parses auditd logs for file access events, command execution tracking, user authentication, and security event analysis.

Part of the [Dargslan Linux Toolkit](https://dargslan.com) - Professional Linux & DevOps tools.

## Installation

```bash
pip install dargslan-audit-log
```

Or install the complete toolkit:

```bash
pip install dargslan-toolkit
```

## Usage

```bash
dargslan-auditlog report
dargslan-auditlog auth
dargslan-auditlog files
```
