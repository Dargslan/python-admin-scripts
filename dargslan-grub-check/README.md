# GRUB Bootloader Checker

Validates GRUB configuration, kernel parameters, boot entries, timeout settings, and default boot options.

Part of the [Dargslan Linux Toolkit](https://dargslan.com) - Professional Linux & DevOps tools.

## Installation

```bash
pip install dargslan-grub-check
```

Or install the complete toolkit:

```bash
pip install dargslan-toolkit
```

## Usage

```bash
dargslan-grub report
dargslan-grub kernels
dargslan-grub params
```
